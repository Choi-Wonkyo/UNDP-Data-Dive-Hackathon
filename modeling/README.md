## 모델링 흐름
모델 목표 : 각 국가와 연도별로 외생 충격 변수, 일반 요인 변수, 그리고 지표별 ODA 지원 금액을 입력받아, 해당 연도부터 2년 후까지 각 지표의 증가율과 감소율을 예측하는 것

- 모델 선택 -> 트리 기반 부스팅 모델 : XGBoost, LightGBM   /     신경망 모델 : MLP(다층 퍼셉트론)
    
- 결측 처리 → x : 그대로 or 평균 대체(MLP)     /   y : 0으로 채우기 or 나라별 평균 대체

- 스케일링 → MLP만 필요 (MinMax)

- Train/Test 분리
    1. **연도 기준 분리** (추천: 기본 전략) → 시계열 흐름 보존됨
        
        
        | 구분 | 연도 | 목적 |
        | --- | --- | --- |
        | **Train** | 2014–2021 | 모델 학습 (가용 데이터 최대 활용) |
        | **Test 1** | 2018–2019 | 정상기 평가 (일반 상황에서 예측력) |
        | **Test 2** | 2020–2021 | 충격기 평가 (팬데믹 등 외란 반응) |
    2. **K-Fold Cross Validation (추천: 보조 실험 전략)**
 
  - 모델 학습 + 평가 →  SHAP(변수 중요도) 분석
